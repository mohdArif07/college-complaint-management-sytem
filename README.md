# College Complaint Management System

## 📌 Description
This project is a Java Swing + MySQL based application used to manage student complaints in a college.

## 🚀 Features
- User Login (Admin & Student)
- Submit Complaint
- View Complaints
- Update Complaint Status (Admin)
- Role-based access (Student/Admin)

## 🛠️ Technologies Used
- Java (Swing)
- MySQL
- JDBC

## ▶️ How to Run
1. Open project in NetBeans
2. Connect MySQL database (projectdb)
3. Run LoginForm.java

## 👤 Users
- Admin → can view & update complaints
- Student → can add & view complaints

## 📂 Project Structure
- src → Java source code
- pom.xml → project dependencies
- MySQL database used

## 🎯 Purpose
This system helps manage complaints efficiently and track their status.
